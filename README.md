# ⚡ Elmentra - The Future of Visual Identity

Elmentra is a high-fidelity, futuristic web platform designed for premium brand management and client project tracking. Built with React, Vite, and React Three Fiber, it combines cutting-edge 3D visuals with a robust administrative backendpowered by Supabase.

---

## 💎 Design Philosophy
Elmentra utilizes a **"Deep Space Modern"** aesthetic, characterized by:
- **Futuristic 3D Elements**: Interactive particle waves and holographic cores.
- **Glassmorphism**: Translucent card surfaces with refined blur effects.
- **Mesh Gradients**: A dynamic background system blending Orange, Sky Blue, and Dark Blue.
- **Mobile-First App Experience**: A dedicated mobile UI with bottom-navigation and fixed headers.

## 🚀 Active Tech Stack
- **Frontend**: [React 19](https://reactjs.org/), [Vite 8](https://vitejs.dev/)
- **3D Graphics**: [React Three Fiber](https://r3f.docs.pmnd.rs/getting-started/introduction), [Three.js](https://threejs.org/)
- **Backend & Auth**: [Supabase](https://supabase.com/) (Real-time DB, Authentication, Storage)
- **Styling**: [Vanilla CSS](https://developer.mozilla.org/en-US/docs/Web/CSS) (Modular & Centralized Design Tokens)
- **Routing**: [React Router DOM v7](https://reactrouter.com/)

## 🛠️ Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18.x or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation
1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/your-username/elmentra.git
    cd elmentra
    ```
2.  **Install Dependencies**:
    ```bash
    npm install
    ```
3.  **Environment Variables**:
    Create a `.env` file in the root directory and add your Supabase credentials:
    ```env
    VITE_SUPABASE_URL=your_supabase_url
    VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
    ```
4.  **Run Development Server**:
    ```bash
    npm run dev
    ```

## ✨ Core Features

### 1. Landing Page (Home)
- **Interactive 3D Hero**: Animated particle wave that reacts to mouse movement.
- **Responsive Navigation**: A premium glassmorphic navbar with mobile slide-drawer.
- **Pricing & Portfolio**: Dynamic 3D hover effects on service cards.

### 2. Client Dashboard
- **Mobile-First Layout**: Bottom navigation bar and mobile top header for native app experience.
- **Real-time Project Tracker**: Visual progress indicators for development milestones.
- **Collaborative Comments**: Inline feedback system with real-time updates from developers.
- **Website Preview/Customize**: Selection of page counts, features, and design preferences.

### 3. Admin Portal
- **Project Oversight**: Manage client accounts, updates, and deadlines.
- **Comment Moderation**: Respond directly to client feedback in real-time.
- **User Management**: Unified dashboard to manage multiple projects simultaneously.
