ELEMENTRA TESTING GUIDE - Desktop & Mobile
===========================================

This guide will help you verify the "Deep Space Modern" design and functional consistency of the Elementra platform across all devices.

---
1. HOW TO ACCESS LOCAL SITE ON MOBILE
---
To test on an actual smartphone:
1. Connect your PC and Phone to the SAME Wi-Fi network.
2. Open your terminal where npm run dev is running.
3. Look for the "Network" URL (e.g., http://192.168.1.XX:5173).
4. Type that exact URL into your mobile browser (Safari/Chrome).

Alternatively, on Desktop:
- Right-click anywhere -> Inspect -> Click the "Toggle Device Toolbar" (Mobile icon) in the top-left of the inspector.

---
2. DESKTOP TESTING CHECKLIST
---
[] BRANDING: Verify "Elementra" logo is visible in Navbar, Footer, and Logins.
[] 3D HERO: Hover over the particle wave to see it react to your cursor.
[] NAVIGATION: Scroll down to see the glassmorphism navbar "shrink" and blur.
[] DASHBOARD (Client):
   - Access via 'Dashboard' link after login.
   - Verify the fixed Sidebar on the left.
   - Test "Customize Your Website" tab - check if WhatsApp link works.
[] DASHBOARD (Admin):
   - Navigate to /admin-login to access.
   - Verify Admin Sidebar and stats layout.
   - Test "Action" buttons in the Project Tracker.

---
3. MOBILE TESTING CHECKLIST
---
[] MOBILE HEADER: Ensure logo is centered and there is a "Logout" icon on the right.
[] BOTTOM NAV: Ensure the 4 main tabs (Overview, Updates, Comments, Customize) are visible at the bottom.
[] ACTIVE TAB: Click tabs and verify the active icon "lifts" slightly with an orange dot.
[] VERTICAL TIMELINE: In Overview, scroll down to see the Progress Tracker. It should be vertical, not horizontal.
[] MENU DRAWER: In the Home page, click the "HiMenu" (3 lines) and verify the premium blurred side-drawer.
[] FORM SCALING: Visit Login/Register pages and ensure inputs are easy to tap.

---
4. FUNCTIONAL FLOW TEST
---
1. Register a new user.
2. Log in and post a "Comment" in the Project Discussion.
3. Open Admin Dashboard and verify the comment appeared.
4. Reply as Admin and check the "Real-time" update on the Client Dashboard.
5. In the Customize tab, change the "Number of Pages" and click "Send on WhatsApp".

---
Report any visual overlapping or layout breaks to the Elementra Team!
